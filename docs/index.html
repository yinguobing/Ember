核心特性

为内容而设计，为阅读而优化

Ember 不止是一个主题，它是你文字的家。每一个细节都经过打磨，只为呈现最好的阅读体验。

动态渐变背景

首页英雄区域采用浮动渐变色块动画，亦可切换为站点封面图背景。支持文字逐字切换特效，可自定义短语与动画速度。

自动暗色模式

根据站点背景色的亮度自动切换明暗主题。深色背景下文字与卡片自动适配，无需手动切换，无需媒体查询。

CJK 排版优化

自动检测中文语言环境，加载 CJK 字体栈，调整字间距与行高，段落两端对齐。让中文阅读舒适自然，不再水土不服。

精选内容轮播

深色主题的精选文章轮播组件，支持自动播放与悬停暂停。左右双栏布局同时展示封面图与文字内容，导航控制直观流畅。

全响应式设计

从手机到宽屏显示器，布局自适应调整。首页三栏网格在小屏自动折叠为单栏，卡片图片使用 srcset 按需加载，兼顾速度与体验。

自定义文章模板

内置全幅与宽幅封面图文章模板，满足不同视觉需求。支持热门标签云、相关文章推荐、社交分享（含微信二维码）等丰富组件。

主题配置

灵活可控，随心定制

所有设置项均可在 Ghost 管理后台直接调整，无需修改代码。

设置项 说明 类型 默认值
navigation_layout 导航栏布局，可选 Logo 居左或居中 选择 Logo on the left
site_background_color 站点背景色，同时决定明暗模式切换 颜色 #ffffff
title_font 标题字体，可选现代无衬线体或优雅衬线体 选择 Modern sans-serif
body_font 正文字体，可选现代无衬线体或优雅衬线体 选择 Elegant serif
hero_background_type 首页英雄区域背景类型：动态渐变或站点封面图 选择 Animated Blobs
hero_scramble_phrases 英雄区域文字特效的展示短语，逗号分隔 文本 探索未知,发现美好,创造价值,分享见解
hero_scramble_speed 文字特效动画速度：快速、正常或慢速 选择 Normal
hero_scramble_horizontal 文字特效水平位置，百分比 (0-100) 文本 50
hero_scramble_vertical 文字特效垂直位置，百分比 (0-100) 文本 0
tag_slugs_for_popular_tags 首页展示的热门标签别名，逗号分隔 文本 （自动按文章数量选取前 10 个）
footer_text 页脚自定义文字，支持 HTML 文本 （留空则使用默认版权信息）
white_publication_logo_for_transparent_header 用于透明或深色导航栏的浅色 Logo 图片 （可选上传）
